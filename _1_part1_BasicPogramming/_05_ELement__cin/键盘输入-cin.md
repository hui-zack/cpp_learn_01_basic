
[键盘输入]
.   [cin函数进行键盘输入](./_1_cin__inputDate/ELe_cinDemo.cpp)
    * cin实现键盘输入:
        格式:
            cin >> 变量名;                   //* 将键盘输入值赋值给变量

        如:
            int i = 0;
            cout << "请输入i的值: " << endl;
            cin >> i;                       //* 键盘输入赋值给i
    * 布尔类型的值, 输入任何非0值都为真
